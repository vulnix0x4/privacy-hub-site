> **TL;DR.** Use Bitwarden. It is free for individuals, open-source, audited, and works on every platform. Family plan is $3.33/month for six people, cheaper than any competitor. The paid tier unlocks TOTP storage and emergency access.

## What this category protects

Your entire login surface. A password manager generates a unique random password per site, stores them encrypted, and auto-fills only on the matching origin (which kills phishing-by-lookalike-domain in one move).

## Hero pick: Bitwarden

Open-source server code you can inspect or self-host (Vaultwarden is the community-maintained compatible server). End-to-end encrypted, zero-knowledge architecture. Apps on iOS, Android, every desktop OS, every major browser. What you give up: the UI is utilitarian next to 1Password's, and self-hosting requires a basic server setup.

## Alternatives

**1Password** — Slickest UI in the category. $2.99/month. Watchtower catches breached credentials nicely. Not open-source, which is a dealbreaker for some.

**Proton Pass** — Free tier with unlimited passwords, built-in SimpleLogin alias support. Good if you already pay for Proton; thinner feature set on its own.

**KeePassXC** — Offline-first, open-source, no cloud. You sync the encrypted `.kdbx` file via Syncthing, iCloud, or whatever. Best for advanced users who distrust any cloud vault.
