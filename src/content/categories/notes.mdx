> **TL;DR.** Use Obsidian. Files are plain Markdown in a local folder. No account. Sync, if you want it, is end-to-end encrypted (Obsidian Sync is $4/month) or you wire up your own via Syncthing, iCloud, or a self-hosted git repo.

## What this category protects

Your thinking. Evernote and Notion store every note in plaintext on their servers. Both have had security incidents. Both can be subpoenaed. A local-first note system means the provider has nothing to hand over, because they never had the content.

## Hero pick: Obsidian

Plain Markdown files in a folder you own. Works offline permanently. Enormous plugin ecosystem. Graph view and backlinks are genuinely useful for long-running research. What you give up: proprietary app code (the free tier has no source access). If that is a dealbreaker, jump to Logseq.

## Alternatives

**Logseq** — Open-source, outliner-first, optional git-backed sync. Best pick if you want fully-FOSS and block-level references.

**Standard Notes** — Cloud-first, end-to-end encrypted, opinionated minimal editor. Good if you want Evernote-style sync without Evernote-style surveillance.

**Joplin** — Open-source, Markdown-based, integrates with Nextcloud, Dropbox, or any WebDAV server for encrypted sync. Desktop and mobile clients.
